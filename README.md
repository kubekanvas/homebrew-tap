# KubeCanvas Homebrew Tap

This repository provides the official Homebrew formula for installing the KubeCanvas CLI on macOS.

## Installation

1. **Tap the repository:**
   ```sh
   brew tap kubekanvas/tap
   ```
2. **Install the CLI:**
   ```sh
   brew install kubekanvas
   ```
